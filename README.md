# CORELASI Web

A Laravel-based learning management system focused on scheduling, assignments, attendance, materials, and grading workflows.

## Overview
This repository contains the web-side implementation of CORELASI, including classroom scheduling, teacher and student data management, attendance records, learning materials, submissions, and assessment features.

## Features
- Classroom and academic year management
- Teacher and student management
- Schedule session management
- Assignment and submission workflow
- Attendance records and teaching journals
- Learning materials and assessment support

## Tech Stack
- Laravel
- PHP
- Blade
- MySQL

## Notes
This repository is kept as a project contribution reference and portfolio artifact.
